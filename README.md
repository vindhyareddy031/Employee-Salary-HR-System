# Employee Salary & HR Analytics Management System

## Project Overview
This project is developed using MySQL to manage employee records, salaries, attendance, and performance.

## Features
- Employee Management
- Department Management
- Salary Calculation
- Attendance Tracking
- Performance Management
- Net Salary Calculation
- SQL Views
- Stored Procedures
- Triggers

## Technologies Used
- MySQL
- MySQL Workbench

## SQL Concepts Used
- DDL
- DML
- Joins
- Aggregate Functions
- Views
- Stored Procedures
- Triggers
